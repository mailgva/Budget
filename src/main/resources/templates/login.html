Приложение Бюджет предназначено для ведения, управления и планирования семейного бюджета.
      Функции:
        - Содержит 2 типа операций: Приход и Расход.
        - Раздел Справочники содержит: виды операций и виды валют.
           После регистрации, пользователь имеет автоматически созданные:
           - несколько видов операций;
           - несколько видов валют (гривны, доллары США, евро).
        - Раздел Статистика - показывает операции по дням, с возможностью фильтрации.
        - Раздел Групповая статистика:
           - показывает сумму по операциям за период, и также может показать:
               * количество операций по виду;
               * процент от общей суммы по типу (белая линия);
               * проценты от максимальной суммы вида (желтая линия);
               * детали для вида;
               * диаграммы: круглая, горизонтальная;
               * вертикальная диаграмма для показа динамики по виду.
        - Для объединения в группу пользователей - в разделе Профиль есть ссылка для присоединения к группе.
        - Смена валюты может осуществляться либо на верхней панели, либо в разделе Профиль.
          Одновременно отображаются данные только по одному виду валюты.
